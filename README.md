# Zolder Route Data Analysis

This repository contains a data analysis script focused on the Circuit Zolder route, created for the Solar Car Team Orna 2. The analysis covers key aspects of the route, including geographic coordinates and elevation data, which are crucial for vehicle optimization, strategic planning, and energy management.

## Project Purpose

- Analyze the Circuit Zolder route for the Solar Car Team
- Include and visualize GPS coordinates and elevation profiles
- Support race strategy, efficiency improvements, and simulation

## Features

- Import and process route data (coordinates, elevation)
- Generate relevant statistics for track analysis
- Visualize altitude profiles and route maps
- Modular design for easy extension
